Guia do Usuário: Sistema Financeiro

Este guia ajudará você a usar o sistema financeiro, incluindo como adicionar, editar, excluir transações e visualizar relatórios financeiros.

1. Tela Inicial

Quando você acessar o sistema, verá as principais funcionalidades na tela inicial:

  • Formulário de Transações: Aqui você pode adicionar transações financeiras, sejam receitas ou despesas.
  • Histórico de Transações: Tabela que exibe todas as transações realizadas, com opções para editar ou excluir.

2. Como Adicionar uma Nova Transação

Para adicionar uma nova transação:

  • Descrição: Informe o nome ou descrição da transação (ex: "Compra de materiais").
  • Valor: Insira o valor da transação. Use números positivos para receitas e negativos para despesas.
  • Data: Selecione a data da transação.
  • Tipo: Escolha se a transação é de entrada (receita) ou saída (despesa).
  • Salvar: Após preencher os campos, clique em "Adicionar" para salvar a transação.
Observação: O sistema verifica se todos os campos são preenchidos corretamente antes de adicionar a transação.

3. Como Editar uma Transação

Para editar uma transação existente:

  • Seleção da Transação: Na lista de transações, encontre a transação que deseja alterar e clique no ícone de edição.
  • Modificar Dados: O sistema abrirá um formulário com os dados da transação. Alterar os campos conforme necessário.
  • Salvar Alterações: Clique em "Salvar" para atualizar os dados da transação.

4. Como Excluir uma Transação

Para excluir uma transação:

  • Seleção da Transação: Encontre a transação na lista e clique no ícone de lixeira.
  • Confirmar Exclusão: O sistema pedirá confirmação antes de excluir permanentemente a transação.

5. Visualização de Relatórios

Você pode visualizar relatórios financeiros na tela principal:

  • Gráfico de Pizza: Exibe a distribuição de receitas e despesas.
  • Gráfico de Linha: Mostra a evolução das receitas e despesas ao longo do tempo.

6. Mensagens e Alertas

O sistema pode mostrar alertas para corrigir erros de preenchimento:

  • "Descrição inválida!" – Certifique-se de que a descrição está correta.
  • "Valor inválido!" – O valor deve ser numérico e maior que zero.
  • "Selecione um tipo de transação!" – Selecione entre receita ou despesa.
  • "Campo excedeu o limite!" – O texto inserido ultrapassou o limite de caracteres.

Conclusão

Este guia forneceu as informações básicas para utilizar o sistema financeiro. Agora, você pode adicionar, editar, excluir transações e visualizar gráficos financeiros para gerenciar suas finanças com mais eficiência.
